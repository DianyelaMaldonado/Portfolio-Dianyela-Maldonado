PROYECTO DREAM TICKETS

Un proyecto desarrollado para desktop, y se basa en una página web de boletos de conciertos, en este caso se utiliza un código para personalizar las entradas con mensajes especiales, así, si vas a regalar el boleto puedas enviar algo que escribiste tú mismo. El programa te cifra el texto que introduzcas y puedes descifrarlo en caso de que lo hayas olvidado. Su finalidad en sí es cifrar y descifrar. En la parte técnica se utilizó el cifrado Caesar, usando lenguages como Vanilla Javascript (métodos como offset, forEach, manejo de DOM), HTML y CSS.

PROYECTO ON RANKEDS

Es una aplicación completamente responsive (mobile, tablet y desktop), donde se consumió data directamente desde el API de RIOT Games: developer.riotgames.com/apis. Se inició con el proceso de pensar el qué quería mostrar allí, en este caso información de personajes de League of Legends, adicional se pensó en implementar filtros para buscar por el tipo de los personajes, lo que llevó a la implementación del lado UX, diseñar la aplicación pensando y entendiendo al usuario (investigación) y maquetación de prototipos. De igual manera se entrevistó a usuarios target para obtener feedback del prototipo por si había algún cambio por hacer antes de iniciar a codear. Se elaboraron historias de usuario en Trello para la implementación de todo lo referente a la app; en cuanto a la parte técnica se usó Javascript, HTML y CSS. Se construyó la aplicación respetando el diseño planeado, uso de promesas, de fetch, forEach, manejo de DOM, entre otros que llevaron a la app a ser finalizada en el tiempo fijado (tres semanas).

PROYECTO CODE-DEV

Es una aplicación completamente Mobile First, Single page application, en la cual se implementó CRUD de información consumiendo servicios de Firestore. Trata de Una red social pensada para la industria Tech, en donde puedes interacturar con otros developers, compartir tus conocimientos y publicar tus ideas en posts. Nuestros usuarios target son aquellas personas dedicadas a la industria de desarrolladores que conozcan cualquier tipo de lenguaje de programación, sin importar su background, solo su interés y deseo de contactar con otras personas y que puedan solicitar ayuda a la comunidad en cuanto a dudas o simplemente compartir lo que están pensando. En la parte técnica se trabajó con Firebase, LocalStorage, Firestore Security rules, Firestore Database, Materialize, MVC. La arquitectura del codigo es 100% modularizado. Es un proyecto colaborativo por lo que usamos Trello para planeación y Github para el control de versiones de código.

PROYECTO BURGER QUEEN

Es una aplicación desarrollada principalmente con React JS para desktop y Ipad-pro. Un restaurante que en esta ocasión tiene por nombre Burger Queen que recién quiere implementar herramientas tecnológicas, nos pasa un requerimiento, en este caso quiere que realicemos una app para facilitar la comunicación entre cocina y meseros. La finalidad es hacer el proceso de los pedidos más eficiente y por lo tanto reducir el tiempo de espera para los clientes, así como una comunicación más ágil entre cocineros y meseros. En la parte técnica Además de React JS, se trabajó con Firebase, LocalStorage, Firestore Security rules, Firestore Database, fue un proyecto colaborativo por lo que usamos Git para el control de versiones y GitHub-Issues-Labels para planeación y código compartido en la nube, Figma para el prototipado.
